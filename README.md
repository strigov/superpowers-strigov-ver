# superpowers-strigov-ver — Claude Code Plugin Marketplace

Personal Claude Code plugin marketplace by strigov.

## Plugins

### superpowers-strigov-ver

Personal fork of [superpowers](https://github.com/obra/superpowers): Sonnet-orchestrated multi-model dev workflow — Opus for judgment (planning, code review), Codex xhigh for plan review, Codex high for implementation. Includes 12 upstream skills and a `/dev` slash command.

## Installation

```
/plugin marketplace add github:strigov/superpowers-strigov-ver
/plugin install superpowers-strigov-ver
/reload-plugins
```

## Requirements

- Claude Code
- [Codex plugin](https://github.com/openai/codex-plugin-cc) installed and authenticated (`!codex login`)
